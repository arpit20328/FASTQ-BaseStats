# FASTQ-BaseStats
Tool to Count A T C G bases from FASTQ file with separate count of Masked Bases "N"

## Usage
```sh
python BaseStats.py <Enter your FASTQ file Path>

Sample Output is as follows:  

Base    Count
A       40160918
C       46209207
G       46098045
T       40245144
